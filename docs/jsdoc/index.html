mongodb-proxy

a node proxy for mongodb

overview

A Proxy and Api pair, for mongo databases.
A Store is created for a collection, used to handle crud operations filtered through custom events giving the ability to intercept, alter or halt the process.
A Api with a slightly different syntax calling internally to stores, is also available, very suitable to be exposed to REST services for client use.
